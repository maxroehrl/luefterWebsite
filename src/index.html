Vom Konzept bis zur Umsetzung ...

... haben wir die Boxen so entwickelt, dass unsere Kunden den maximalen Fotospaß haben.

Mittels modernster Technik entstanden so zwei kompakte Fotoboxen, mit denen man nicht nur Sofortfotos anfertigen kann, sondern diese auch noch in einer digitalen Bildergalerie vom Tablet abrufen kann. Hierfür eröffnet die Box automatisch ein eigenes WLAN-Netzwerk, über welches dann die Bilder angesehen werden können.

Die Aufnahme der Bilder kann sowohl im "normalen" Einzelbild-Modus, als auch im Multibild-Modus erfolgen. Außerdem verfügt unsere Box über eine Option zum Aufzeichnen von animierten Bildern, sogenannte GIFs, welche einzigartige Party-Momente in Dauerschleife festhalten.

Natürlich muss für das optimale Foto auch die Beleuchtung stimmig sein, weswegen wir integrierte Leuchten an unseren Fotoboxen haben. Diese sorgen auch bei schwierigen Lichtverhältnissen für eine gute Auslichtung und können ganz nebenbei zum Partylicht umfunktioniert werden.

  • Multibild-, Einzelbild- und GIF-Modus
  • Unkomplizierter Sofortdruck
  • Digitale Bildergalerie mit WiFi
  • Integrierte Beleuchtung
  • Fernsteuerbar mit K&R Remote
